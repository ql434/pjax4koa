pjax4koa
================

> pjax for koa.

## Related module
- [`cheerio`](https://www.npmjs.com/package/cheerio)

## Install
>Install with [npm](https://github.com/npm/npm)

```
$ npm install pjax4koa
```

## Features

## Usage with [pjax4koa](https://github.com/alexmingoia/koa-router)

```js
var app      = require('koa')(),
    pjax4koa = require('pjax4koa');

app.use(pjax4koa());
```
